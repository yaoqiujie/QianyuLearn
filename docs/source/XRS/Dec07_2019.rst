运算律

1. 乘法交换律

    3 x 7 = 7 x 3

    a x b = b x a

2. 乘法结合律

    9 x 2 x 5 = 9 x (2 x 5)

    a x b x c = a x (b x c)

3. 乘法分配律

    25 x (4 + 10) = 25 x 4 + 25 x 10

    a x (b + c) = a x b + a x c

    a x (b - c) = a x b - a x c
