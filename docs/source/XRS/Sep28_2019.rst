1. 一笔画
=================

要求
    笔不离纸、线不重复、一笔画完

判断
    #. 连通图
    #. 奇点个数
        2个奇点: 奇进奇出

        0个奇点: 同进同出

2. 多笔画
================

最少几笔完成
    0个奇点一笔完成

    奇点个数除以2

多变一(消灭奇点)
    两个奇点之间加线或者去线

3. 应用
================

实物图变点线图
    1. 区域(可重复) --> 点
    2. 通道(不可重复) --> 线
