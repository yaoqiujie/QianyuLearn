题型
===========

1. 二量和倍
^^^^^^^^^^^^^^^^^^^

    **整倍:     找和对应的份数**

    **非整倍:   多砍少补变整数**

2. 三量和倍
^^^^^^^^^^^^^^^^^^

    **找最小的一倍数**

方法
==========

1. 线段图
^^^^^^^^^^^^^^^^^^

    #. 先画一倍数(少的)
    #. 左端堆砌，再画多倍数
    #. 多: 再加线, 少: 变虚线
    #. 标数: 数与图要对应

特点
============

    **给来给去和不变**
