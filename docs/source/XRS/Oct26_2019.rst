2019年10月26日 以半击倍(差倍初步)
######################################

题型
================

1. 基本型
^^^^^^^^^^^^^
    * 整倍型
        找 **差** 对应的倍数

    * 非整倍型
        **多去少补** 变整倍

2. 暗差型
^^^^^^^^^^^^^^

    **你增我减: 差相加**

    **同增同减: 差不变**

方法
==================

    *画线段图*

    #. 先画一倍数(少的)
    #. 再画多倍数
    #. 最后找对应差
